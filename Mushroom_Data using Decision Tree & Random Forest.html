Machine Learning Process¶

  • Clean/Prepare your data (EDA Process)
    • Missing value treatment
    • outlier removal
    • Standardization
    • Dummy variables conversion
  • Split your data in training & testing (train, test)
  • Separate input & output cols (train_x, train_y, test_x, test_y)
  • Build Model using training data
  • Predict output values for test data
  • Evaluate your model
    • Accuracy, Sensitivity, Specificity
  • Fine tune your model(s) for better performance
    • Hyper parameter tuning, cross validation

Predicting using Decision Tree¶

In [52]:
mush_dummy = pd.get_dummies(mush.drop('Class', axis=1))
mush_dummy['Class'] = mush['Class']

train,test = train_test_split(mush_dummy, test_size=0.3)
train_x = train.drop('Class', axis=1)
train_y = train['Class']

test_x = test.drop('Class', axis=1)
test_y = test['Class']

model = DecisionTreeClassifier(random_state=100)
model.fit(train_x, train_y)

test_pred = model.predict(test_x)
df_pred = pd.DataFrame({'actual': test_y,
                       'predicted': test_pred})

df_pred['pred_status'] = df_pred['actual']==df_pred['predicted']
df_pred['pred_status'].sum()/df_pred.shape[0]*100

def draw_tree(model, columns):
    
    import pydotplus
    from sklearn.externals.six import StringIO
    from IPython.display import Image

    import os
    graphviz_path = 'C:\Program Files (x86)\Graphviz2.38/bin/'
    os.environ["PATH"] += os.pathsep + graphviz_path
    
    dot_data = StringIO()
    tree.export_graphviz(model,out_file=dot_data,
                    feature_names=columns)
    graph = pydotplus.graph_from_dot_data(dot_data.getvalue())
    return Image(graph.create_png())
